# Database Schema

Repo ini tidak memiliki database schema.

`ai-skills` adalah distribusi file skill, installer shell, metadata release, dan dokumentasi. Jika project nanti menambahkan database, dokumentasikan schema di halaman ini dengan:

- daftar tabel atau collection,
- primary key dan foreign key,
- index penting,
- migration file terkait,
- aturan kompatibilitas data saat upgrade dan rollback.
